画面一覧

/login
ログイン画面
ユーザー名・パスワード入力。認証後はホームへリダイレクト。
/
ホーム画面
ログイン後のトップ。ナビゲーションメニュー。
/tables
テーブル一覧画面
業務テーブル（5種）の一覧。全テーブルExcelエクスポートボタン。
/tables/:name
テーブル詳細画面
データ一覧。ページング・検索・ソート・追加・編集・削除・Excel出力。
/tables/:name/edit/:id
行編集画面
既存レコードの編集フォーム。
/tables/:name/edit/:id/confirm
編集確認画面
変更内容の確認・確定。
/tables/:name/import
CSVインポート画面
単テーブルへのCSVファイル取込。
/users
ユーザー管理画面
ユーザーCRUD。ADMIN のみアクセス可。
/masked-columns
カラムマスク設定画面
ロール別カラム非表示設定の管理。ADMIN のみ。
/csv-bulk
CSV一括入出力画面
全テーブル一括エクスポート/インポート。

画面遷移図

[/login]
    │ 認証成功
    ▼
[/] ホーム
    ├─ テーブル管理 ──→ [/tables] テーブル一覧
    │                        └─ テーブル選択 ──→ [/tables/:name] データ一覧
    │                                                  ├─ 編集ボタン ──→ [/tables/:name/edit/:id]
    │                                                  │                    └─ 確認 ──→ [/tables/:name/edit/:id/confirm]
    │                                                  └─ インポート ──→ [/tables/:name/import]
    ├─ ユーザー管理 ──→ [/users]      （ADMIN のみ）
    ├─ マスク設定  ──→ [/masked-columns]  （ADMIN のみ）
    └─ CSV一括    ──→ [/csv-bulk]

    ※ 認証なし、またはトークン期限切れ → [/login] へリダイレクト

API エンドポイント一覧

認証

メソッドパス権限説明
POST /api/auth/login 全員 ログイン。JWTトークンを返す。

ユーザー管理

メソッドパス権限説明
GET/api/usersADMINユーザー一覧取得
POST/api/usersADMINユーザー作成
PUT/api/users/{id}ADMINユーザー更新（パスワード・ロール変更）
DELETE/api/users/{id}ADMINユーザー削除

テーブルデータ操作

メソッドパス権限説明
GET/api/tables全員業務テーブル一覧取得
GET/api/tables/{name}全員テーブルデータ取得（ページング・検索・ソート）
GET/api/tables/{name}/columns全員カラム一覧取得
POST/api/tables/{name}/rowsEDITOR+行追加
PUT/api/tables/{name}/rows/{id}EDITOR+行更新
DELETE/api/tables/{name}/rows/{id}EDITOR+行削除

カラムマスク

メソッドパス権限説明
GET/api/masked-columns全員マスク設定一覧
POST/api/masked-columnsADMINマスク設定追加
DELETE/api/masked-columns/{id}ADMINマスク設定削除

CSV / Excel インポート・エクスポート

メソッドパス権限説明
POST/api/import/{tableName}EDITOR+CSVインポート（単テーブル）。mode=overwrite は ADMIN のみ。
GET/api/export/allEDITOR+全テーブル一括CSVエクスポート
POST/api/import/allEDITOR+全テーブル一括CSVインポート
GET/api/export/excel/{tableName}EDITOR+Excelエクスポート（単テーブル）
GET/api/export/excel/allEDITOR+全テーブルExcel一括エクスポート

リクエスト/レスポンス例

POST /api/auth/login

リクエスト:

{ "username": "admin", "password": "password123" }

レスポンス (200 OK):

{ "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..." }

レスポンス (401 Unauthorized):

{ "error": "ユーザー名またはパスワードが正しくありません" }

GET /api/tables/{name}（クエリパラメータ）

パラメータ型説明デフォルト
pageintページ番号（0始まり）0
sizeint1ページあたり件数20
searchstring全カラム対象キーワード検索—
sortColstringソート対象カラム名—
sortDirstringasc / descasc

レスポンス:

{
  "data": [ { "ANKEN_ID": 1, "ANKEN_NAME": "サンプル案件", ... }, ... ],
  "total": 120,
  "page": 0,
  "size": 20
}

POST /api/import/{tableName}（CSVインポート）

リクエスト: multipart/form-data

フィールド型説明
fileMultipartFileUTF-8 CSVファイル
modestringappend（デフォルト）/ overwrite

レスポンス:

{ "inserted": 15, "errors": ["行3: インポートエラーが発生しました"] }

共通エラーレスポンス

HTTPステータスボディ例発生条件
400 Bad Request{ "error": "対象外のテーブルです" }allowlist外テーブル指定
401 Unauthorized（レスポンスボディなし）トークン未送信・期限切れ
403 Forbidden{ "error": "権限がありません" }ロール不足
500 Internal Server Error{ "error": "処理中にエラーが発生しました" }DB障害等（詳細はサーバーログ）

共通リクエストヘッダー

ヘッダー値必須
AuthorizationBearer <JWTトークン>ログイン以外の全エンドポイント
Content-Typeapplication/jsonJSONリクエスト時
