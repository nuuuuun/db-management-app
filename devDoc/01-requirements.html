R-01 システム要件定義

システム概要

社内業務データ（案件・要求・要件・アプリ・環境）を一元管理するWebアプリケーション。
社内LAN環境に設置したサーバーで動作し、15名程度のユーザーが同時利用する。ブラウザのみで操作可能。

システム構成

区分技術バージョン役割
バックエンドSpring Boot3.4.5REST API サーバー
フロントエンドVue.js3.5.32SPA (Single Page App)
データベースPostgreSQL16業務データ永続化
コンテナDocker / Docker Compose最新安定版本番・開発環境の統一
認証JWT + Spring Security—ステートレス認証
JavaOpenJDK17 LTSバックエンドランタイム

利用環境

項目内容
ネットワーク社内LAN（インターネット非公開）
同時接続数最大15名程度
サーバーOSLinux（Docker動作可能環境）
クライアントWebブラウザ（Chrome / Edge 最新版推奨）
アクセスURLhttp://[サーバーIP]:18080

R-02 機能要件一覧

認証・ユーザー管理

ID機能優先度概要
F-01ログイン必須ユーザー名・パスワードでJWTトークン取得。24時間有効。
F-02ユーザー一覧必須登録ユーザーの一覧表示。ADMIN のみ操作可。
F-03ユーザー作成必須新規ユーザー登録。ADMIN のみ。パスワードはBCryptでハッシュ化。
F-04ユーザー編集必須パスワード・ロール変更。ADMIN のみ。
F-05ユーザー削除必須ユーザー削除。ADMIN のみ。

データ管理（テーブル操作）

ID機能優先度概要
F-10テーブル一覧必須業務テーブル（5種）の一覧表示。
F-11データ一覧必須テーブルデータのページング表示・検索・ソート。
F-12行追加必須新規レコード登録。ADMIN/EDITOR のみ。
F-13行編集必須既存レコード更新（確認画面あり）。ADMIN/EDITOR のみ。
F-14行削除必須レコード削除。ADMIN/EDITOR のみ。

カラムマスク設定

ID機能優先度概要
F-20マスク設定一覧必須ロール別カラム非表示設定の一覧表示。
F-21マスク設定追加必須指定テーブル・カラムをロール別に非表示設定。ADMIN のみ。
F-22マスク設定削除必須マスク設定の解除。ADMIN のみ。

インポート・エクスポート

ID機能優先度概要
F-30CSVインポート（単体）必須テーブル単位のCSVファイル取込。追加/上書きモード選択可。
F-31CSV一括エクスポート必須全業務テーブルを1つのCSVファイルへ出力。ロール別マスク適用。
F-32CSV一括インポート必須一括エクスポートフォーマットのCSVファイルを全テーブルへ取込。
F-33Excelエクスポート（単体）推奨テーブル単位のExcelファイル出力。
F-34Excel一括エクスポート推奨全業務テーブルをシート分けしたExcelファイルへ出力。

操作履歴

ID機能優先度概要
F-40操作履歴記録必須INSERT/UPDATE/DELETE 操作時に操作者・日時・変更前後を自動記録。

R-03 非機能要件定義

パフォーマンス

項目目標値備考
ページ表示レスポンス3秒以内LAN内通常操作時
CSV出力（全テーブル）10秒以内1000件規模想定
同時接続15名社内LAN運用

可用性

項目要件
稼働時間業務時間内（平日9:00〜18:00）
計画停止事前周知の上で週末メンテナンス可
データ永続性Dockerボリュームによるデータ永続化。本番では定期バックアップ必須。

セキュリティ

項目要件
認証JWT認証。トークン有効期限24時間。パスワードはBCryptハッシュ化。
認可3ロール（ADMIN/EDITOR/VIEWER）によるRBAC。
シークレット管理DBパスワード・JWT秘密鍵は .env ファイル管理（Gitコミット禁止）。
テーブルアクセス制御業務テーブルallowlistにより管理テーブルへの直接アクセスを禁止。
カラム非表示ロール別カラムマスク設定によるデータ保護。

保守性

項目要件
設定の外部化接続情報・秘密鍵は環境変数で切り替え可能。
ログ操作履歴はDBに記録。エラーはサーバーログへ出力。
デプロイdocker compose up --build で1コマンドデプロイ可能。

R-04 ユーザー・権限要件定義

ロール定義

ロール対象者概要
ADMINシステム管理者全機能利用可。ユーザー管理・マスク設定・上書きインポートが可能。
EDITOR業務担当者業務データのCRUD操作が可能。ユーザー管理・上書きインポートは不可。
VIEWER閲覧者データ参照のみ。エクスポートも不可。

機能別権限マトリクス

機能ADMINEDITORVIEWER
ログイン✅✅✅
データ参照✅✅✅
データ追加・編集・削除✅✅❌
CSVインポート（追加）✅✅❌
CSVインポート（上書き）✅❌❌
CSV/Excelエクスポート✅✅❌
ユーザー管理✅❌❌
カラムマスク設定✅❌❌

カラムマスク仕様

  • ADMIN は常に全カラムを参照可能
  • EDITOR / VIEWER それぞれに対して非表示カラムを個別設定可能
  • VIEWER にマスク設定したカラムは、EDITOR も自動的に非表示
  • マスクは一覧表示・エクスポート・CSV一括出力に適用される
